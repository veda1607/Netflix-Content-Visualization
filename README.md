# Netflix Content Visualization

## Project Overview

This project analyzes Netflix Movies and TV Shows data using Python, Google Colab, Power BI, and data visualization techniques.

## Objectives

* Perform data cleaning
* Analyze Netflix content trends
* Visualize content distribution
* Build an interactive Power BI dashboard

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab
* Power BI
* GitHub

## Dataset

Netflix Movies and TV Shows Dataset from Kaggle.

## Dashboard Features

* Total Content
* Movies vs TV Shows
* Rating Distribution
* Top Countries
* Content by Release Year
* Interactive Filters
